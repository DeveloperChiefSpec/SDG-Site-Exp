    Նպատակը 3: Ապահովել առողջ կյանք և խթանել բարեկեցություն բոլորի համար՝ անկախ տարիքից

    1. Գլխավոր էջ
    2. Նպատակը 3

    Թիրախներ

    Ցուցանիշներ

    3.1 Մինչև 2030թ. կրճատել մայրական մահացության համամասնությունը մինչև 70¬ից ցածր՝ 100 հազար կենդանածնի հաշվով
    3.1.1 Առցանց հաշվետվություն Մայրական մահերը 100 հազար կենդանածնի հաշվով
    3.1.1.a Առցանց հաշվետվություն Մայրական մահերը 100 000 կենդանածնի հաշվով (միջին եռամյա ցուցանիշ)
    • Ազգային
    3.1.2 Առցանց հաշվետվություն Հմուտ բժշկական անձնակազմի կողմից ընդունվող ծնունդների համամասնությունը
    3.2 Մինչև 2030թ. վերջ դնել նորածինների և մինչև 5 տարեկան երեխաների կանխարգելելի մահերին. բոլոր երկրները պետք է նպատակ ունենան նվազեցնել վաղ նորածնային մահացությունը առնվազն մինչև 12¬ից ցածր՝ 1000 կենդանածնի հաշվով, իսկ մինչև 5 տարեկան երեխաների մահացությունը՝ առնվազն մինչև 25¬ի՝ 1000 կենդանածնի հաշվով
    3.2.1 Առցանց հաշվետվություն Մինչև 5 տարեկան երեխաների մահացության ցուցանիշը
    3.2.2 Առցանց հաշվետվություն Վաղ նորածնային մահացության ցուցանիշը
    3.2.2.a Առցանց հաշվետվություն Մանկական մահացության ցուցանիշը
    • Ազգային
    3.3 Մինչև 2030թ. վերջ դնել ՁԻԱՀ¬ի, տուբերկուլյոզի, մալարիայի և անտեսված արևադարձային հիվանդությունների համաճարակներին և պայքարել հեպատիտի, ջրի միջոցով փոխանցվող հիվանդությունների և այլ վարակիչ հիվանդությունների դեմ
    3.3.1 Առցանց հաշվետվություն Նոր ՄԻԱՎ վարակների թիվը 1000 չվարակված բնակչի հաշվով ըստ սեռի, տարիքի և բնակչության հիմնական խմբերի
    3.3.1.a Առցանց հաշվետվություն Կրծքով չկերակրող ՄԻԱՎ վարակ ունեցող մայրերից երեխային ՄԻԱՎ-ի փոխանցման ցուցանիշ
    • Ազգային
    3.3.1.b Առցանց հաշվետվություն Իրենց կարգավիճակի մասին իմացող ՄԻԱՎ-ով ապրող մարդկանց տոկոսը
    • Ազգային
    3.3.1.c Առցանց հաշվետվություն Իրենց կարգավիճակի մասին իմացող ՄԻԱՎ-ով ապրող մարդկանց տոկոսը, որոնք ստանում են ՀՌՎ բուժում
    • Ազգային
    3.3.1.d Առցանց հաշվետվություն ՀՌՎ բուժում ստացող ՄԻԱՎ վարակով պացիենտների տոկոսը, որոնք ունեն վիրուսային ծանրաբեռնվածության (ՎԾ) չհայտնաբերվող մակարդակ
    • Ազգային
    3.3.2 Առցանց հաշվետվություն Տուբերկուլյոզի 1000 մարդու հաշվով նոր դեպքերի թիվը մեկ տարվա ընթացքում
    3.3.2.a Առցանց հաշվետվություն Բազմադեղակայուն տուբերկուլյոզով հիավանդացության ցուցանիշ (100 000 բնակչի հաշվով)
    • Ազգային
    3.3.3 Առցանց հաշվետվություն Մալարիայի 1000 մարդու հաշվով նոր դեպքերի թիվը մեկ տարվա ընթացքում
    3.3.3.a Առցանց հաշվետվություն Մալարիայի տեղական դեպքերի թիվը մեկ տարվա ընթացքում
    • Ազգային
    3.3.4 Առցանց հաշվետվություն Հեպատիտ Բ¬ով վարակման դեպքերի թիվը 100 հազար բնակչի հաշվով
    3.3.4.a Առցանց հաշվետվություն 0-18 տարեկան սուր հեպատիտ Բ-ով հիվանդացությունը (100 000 բնակչի հաշվով)
    • Ազգային
    3.3.4.b Առցանց հաշվետվություն Հեպատիտ Բ-ով առանձին կամ համակցված բաղադրիչով 3 դեղաչափ պատվաստումներումների ընդգրկվածություն 95 և ավել ընդգծված բաղադրիչով
    • Ազգային
    3.3.5 Ուսումնասիրել տվյալների աղբյուրները Անտեսված արևադարձային հիվանդությունների դեմ միջամտությունների կարիք ունեցող մարդկանց թիվը
    3.3.5.a Առցանց հաշվետվություն Լեյշմանիոզով հիվանդների թիվ
    • Ազգային
    3.3.5.b Առցանց հաշվետվություն Լեյշմանիոզից մահվան դեպքեր
    • Ազգային
    3.4 Մինչև 2030թ. մեկ երրորդով կրճատել վաղաժամ մահացությունը ոչ վարակիչ հիվանդություններից՝ կանխարգելման և բուժման միջոցով, և խթանել հոգեկան առողջությունը և բարեկեցությունը
    3.4.1 Առցանց հաշվետվություն Սրտանոթային հիվանդություններին, քաղցկեղին, շաքարային դիաբետին կամ շնչուղիների քրոնիկական հիվանդություններին վերագրվող մահացությունների թիվը
    3.4.2 Առցանց հաշվետվություն Ինքնասպանության հետևանքով մահացության ցուցանիշ
    3.5 Ուժեղացնել թմրամիջոցների չարաշահման կանխարգելումը և բուժումը, այդ թվում՝ թմրադեղերի չարաշահումը և ոգելից խմիչքների վնասաբեր օգտագործումը
    3.5.1 Ուսումնասիրել տվյալների աղբյուրները Թմրամիջոցների օգտագործման հետևանքով առաջացած խանգարումների բուժման միջամտությունների (դեղորայքային բուժում, սոցիալ-հոգեբանական, վերականգնողական և հետբուժական խնամքի ծառայություններ) ընդգրկումը
    3.5.1.a Առցանց հաշվետվություն Թմրամոլության տառապող հիվանդների թիվը
    • Ազգային
    3.5.2 Առցանց հաշվետվություն Ալկոհոլի վնասաբեր օգտագործումը, որն ըստ ազգային համատեքստի սահմանված է որպես ալկոհոլի սպառում մեկ շնչի հաշվով (15 և ավելի բարձր տարիքի անձանց շրջանում) մեկ օրացուցային տարվա ընթացքում՝ մաքուր սպիրտի լիտրերով
    3.6 Մինչև 2020թ. աշխարհում կիսով չափ կրճատել ճանապարհատրանսպորտային պատահարներից մահերի և վիրավորվելու դեպքերի թիվը
    3.6.1 Առցանց հաշվետվություն Ճանապարհատրանսպորտային պատահարներից մահացության թիվը
    3.7 Մինչև 2030թ. ապահովել սեռական և վերարտադրողական առողջության պահպանման ծառայությունների համընդհանուր հասանելիություն, այդ թվում՝ ընտանիքի պլանավորման, տեղեկատվության և կրթության, ինչպես նաև վերարտադրողական առողջության ինտեգրումը ազգային ռազմավարություններում և ծրագրերում
    3.7.1 Առցանց հաշվետվություն Վերարտադրողական տարիքի (15¬49 տարեկան) կանանց համամասնությունը, որոնց ընտանիքի պլանավորման կարիքները բավարարվում են ժամանակակից մեթոդներով
    3.7.2 Առցանց հաշվետվություն Դեռահասների (10-14 տարեկան, 15-19 տարեկան) ծնելիությունը տվյալ տարիքային խմբի 1000 կնոջ հաշվով
    3.7.2.a Առցանց հաշվետվություն Դեռահասների (15-17 տարեկան) ծնելիությունը տվյալ տարիքային խմբի 1 000 կնոջ հաշվով
    • Ազգային
    3.8 Հասնել առողջապահության համընդհանուր ընդգրկմանը, այդ թվում՝ պաշտպանություն ֆինանսական ռիսկերից, որակյալ էական առողջապահական ծառայությունների հասանելիություն և անվտանգ, արդյունավետ, որակյալ և մատչելի կարևոր դեղորայքի և պատվաստանյութերի հասանելիություն բոլորի համար
    3.8.1 Ուսումնասիրել տվյալների աղբյուրները Հիմնական առողջապահական ծառայությունների ընդգրկումը (հետագծող միջամտությունների հիման վրա սահմանված հիմնական ծառայությունների միջին ընդգրկումը ներառյալ վերարտադրողական, մայրական, նորածնի և մանկական առողջության, վարակիչ հիվանդությունների, ոչ վարակիչ հիվանդությունների ու ծառայության կարողունակությունն ու մատչելիությունը հանրության և բնակչության ամենախոցելի խավի համար)
    3.8.1.a Առցանց հաշվետվություն Որակավորված մասնագետի կողմից նախածննդյան հսկողության և խնամքի ապահովում
    • Ազգային

    3.9 Մինչև 2030թ. էականորեն կրճատել վտանգավոր քիմիական նյութերի և օդի, ջրի և հողի աղտոտման և թունավորման հետևանքով մահերի և հիվանդությունների թիվը
    3.9.1 Ուսումնասիրել տվյալների աղբյուրները Շենքերի ներսում և դրսում օդի աղտոտմանը վերագրվող մահացության ցուցանիշը
    3.9.1.a Առցանց հաշվետվություն Գրանցված սոցիալ-կենցաղային թունավորման դեպքերից զոհերի թվաքանակը
    • Ազգային
    3.9.2 Առցանց հաշվետվություն Աղտոտված ջրի, անապահով սանիտարական պայմանների ու հիգիենայի իսպառ բացակայության պատճառով մահացությունների թիվը (անապահով «Ջուր և սանիտարահիգիենիկ պայմաններ բոլորի համար» (WASH) ծառայություններին ենթարկված լինելը)
    3.9.3 Առցանց հաշվետվություն Ոչ դիտավորյալ թունավորումների պատճառով մահացության թիվը
    3.b Օժանդակել պատվաստանյութերի և դեղորայքի հետազոտություններին և զարգացմանը վարակիչ և ոչ վարակիչ հիվանդությունների դեմ, որոնցից հիմնականում տուժում են զարգացող երկրները, հասանելի դարձնել մատչելի էական դեղորայքը և պատվաստանյութերը համաձայն Մտավոր սեփականության իրավունքների առևտրային ասպեկտներին վերաբերող (TRIPS) համաձայնագրի և հանրային առողջապահության վերաբերյալ Դոհայի հռչակագրի, որը հաստատում է զարգացող երկրների իրավունքը՝ լիարժեք կերպով օգտագործե¬լու Համաձայնագրի՝ մտավոր սեփականության իրավունքների` առևտրին առնչվող հայեցակետերի մասին դրույթները, որոնք վերաբերում են հանրային առողջության պահպանման ճկուն եղանակներին, և, մասնավորապես, հասանելի դարձնել դեղորայքը բոլորի համար
    3.b.1.a Առցանց հաշվետվություն Պատվաստումներում ընգրվածության ցուցանիշ նպատակային քանակակազմերի շրջանում 1տ
    • Ազգային

    3.b.3 Ուսումնասիրել տվյալների աղբյուրները Բժշկական հաստատությունների մասնաբժինը, որոնք մշտապես ունեն անհրաժեշտ և մատչելի դեղորայք
